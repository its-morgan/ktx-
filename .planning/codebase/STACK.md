# Tech Stack

## Runtime & Frameworks
- **Language:** PHP ^8.1
- **Framework:** Laravel ^10.10
- **Platform:** Windows (Laragon)

## Database
- **Driver:** MySQL (Standard Laravel setup)
- **ORM:** Eloquent

## Frontend
- **CSS:** Tailwind CSS ^3.1
- **JS Framework:** Alpine.js ^3.4
- **UI Components:** Flowbite ^1.6
- **Asset Bundler:** Vite ^5.0

## Key Packages
- `laravel/sanctum`: Token-based API authentication
- `laravel/breeze`: Minimal authentication scaffolding
- `guzzlehttp/guzzle`: HTTP client
- `axios`: Promise based HTTP client for the browser
