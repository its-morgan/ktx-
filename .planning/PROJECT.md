# Project: Dormitory Management System (hethongquanlyktx)

## Vision
A comprehensive web-based management system for dormitories, providing tools for administrators to manage facilities and students, and a portal for students to register and track their stay.

## Core Features
- **Student Management:** Profiles, registration, room assignment.
- **Room Management:** Types, equipment (Vattu), status.
- **Contract Management:** Generation, approval, termination.
- **Financials:** Invoices, payments.
- **Notifications:** Announcements and individual alerts.

## Technology Stack
- **Backend:** Laravel 10 (PHP 8.1)
- **Frontend:** Tailwind CSS, Alpine.js, Flowbite
- **Database:** MySQL
- **Workflow:** Get Shit Done (GSD) AI Framework

## Constraints
- **Localization:** All user-facing text must be in Vietnamese (accented).
- **Security:** CSRF protection, middleware-based role authentication.
